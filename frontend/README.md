# 🎓 LearnX | Frontend Application

This directory contains the static client-side web application for the LearnX Learning Management System (LMS).

## 🚀 Quick Links
- **Live Site:** [https://nitishkpathak.github.io/LearnX/frontend/](https://nitishkpathak.github.io/LearnX/frontend/)
- **Main Documentation:** Please refer to the [Root README.md](../README.md) for full installation and system documentation.

## 🛠️ Tech Stack & Libraries
- **Structure & Styling:** HTML5, Vanilla CSS3 (Custom variables, glassmorphic dark/light styles), Bootstrap 5
- **Controllers & Routing:** Vanilla ES6+ JavaScript, custom single-page router (`router.js`)
- **Modals & Alerts:** SweetAlert2 (Rich animations)
- **Icons:** FontAwesome 6, Bootstrap Icons (CDN)
- **PDF Generation:** `html2canvas` (v1.4.1), `jsPDF` (v2.5.1)

## 📂 Key Files
- `index.html`: Modern, responsive corporate landing page.
- `dashboard.html`: Student workspace dashboard with course player, quizzes, assignments, leaderboard, and badges.
- `assets/js/dashboard.js`: Orchestrates dynamic course rendering, playable syllabus, and PDF generation.
- `assets/js/router.js`: Custom router that manages navigation state, login checks, and multi-page redirection.
- `assets/js/theme.js`: Bi-directional light/dark theme sync engine.
